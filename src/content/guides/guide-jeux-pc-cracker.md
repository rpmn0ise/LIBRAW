## Les deux types de jeux crackés

**Repack** — le jeu est compressé au maximum pour réduire la taille du téléchargement. Il faut passer par une étape d'installation et de décompression, qui peut prendre du temps selon la taille du jeu et les performances du PC. Idéal si la connexion est limitée.

**Portable** — le jeu est pré-installé dans un dossier compressé. On télécharge, on extrait, on lance. Pas d'installation, pas d'étapes intermédiaires. C'est la méthode la plus rapide et la moins sujette aux erreurs. Ankergames est la principale source pour ce type de jeux.

## Avant de commencer

### uBlock Origin — obligatoire

Les sites de téléchargement illégal sont truffés de publicités agressives, de faux boutons de téléchargement et de redirections vers des pages malveillantes. Sans bloqueur de pub, le risque de télécharger un malware à la place du jeu est réel.

Installe [uBlock Origin](https://ublockorigin.com) sur ton navigateur avant de faire quoi que ce soit d'autre.

### Client torrent

Si tu veux télécharger en torrent, il te faut [qBittorrent](https://www.qbittorrent.org/) — open-source, sans pub, sans adware. Évite uTorrent et BitTorrent (adware).

Un VPN supportant le port forwarding est recommandé pour le torrent (Mullvad, ProtonVPN). Pour le DDL (téléchargement direct), aucun VPN n'est nécessaire.

### Créer un dossier exclu de l'antivirus

Windows Defender va supprimer des fichiers de crack pendant le téléchargement ou la décompression. Pour éviter ça, crée un dossier dédié à tes jeux et exclus-le de la surveillance de l'antivirus avant de commencer.

**Paramètres Windows → Confidentialité et sécurité → Sécurité Windows → Protection contre les virus et menaces → Gérer les paramètres → Exclusions → Ajouter une exclusion → Dossier**

Sélectionne ton dossier de jeux. Tout ce qui s'y trouve sera ignoré par Defender.

### Pourquoi l'antivirus s'affole

Les cracks modifient le code original du jeu pour contourner la protection DRM. Cette modification ressemble structurellement à ce que font certains malwares — d'où les faux positifs. Ce n'est pas forcément dangereux, mais ça mérite de télécharger uniquement depuis les sources listées ci-dessus.

## Télécharger un repack

### Trouver le bon lien

Sur Google, cherche le nom du jeu + le nom du site. Exemple : `Elden Ring FitGirl Repacks`. Vérifie que le domaine dans la barre d'adresse correspond exactement au domaine officiel du site — les sites pirates sont souvent copiés avec des domaines très proches.

Sur la page du jeu, le lien de téléchargement officiel est en général discret. Les gros boutons colorés "DOWNLOAD NOW" sont presque toujours des pubs. Cherche un lien textuel ou un bouton sobre.

### DDL avec JDownloader 2

Pour le téléchargement direct (DDL), [JDownloader 2](https://jdownloader.org/) centralise les liens et gère automatiquement les hôtes de fichiers (1fichier, Mediafire, Gofile...).

1. Télécharge et installe JDownloader 2
2. Lors de l'installation, décoche les logiciels supplémentaires proposés
3. Copie les liens DDL depuis la page du jeu
4. JDownloader les détecte automatiquement et les ajoute à la file
5. Lance le téléchargement

### Torrent

Télécharge le fichier `.torrent` ou copie le lien magnet depuis la page du jeu, ouvre-le dans qBittorrent. Assure-toi que ton VPN est actif avant de lancer le téléchargement.

## Redistribuables — si le jeu ne se lance pas

La plupart des jeux nécessitent des composants système installés séparément. Si un jeu refuse de démarrer, installe ces redistribuables :

- [DirectX](https://www.microsoft.com/fr-fr/download/details.aspx?id=35)
- [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) (le plus récent suffit généralement)
- [.NET Runtime](https://dotnet.microsoft.com/fr-fr/download)

Si le jeu refuse toujours de démarrer, essaie de lancer l'exécutable **en tant qu'administrateur** (clic droit → Exécuter en tant qu'administrateur).

## Les erreurs classiques

**L'antivirus a supprimé un fichier** — crée l'exclusion du dossier dans Defender, puis re-télécharge le fichier manquant ou relance l'installation depuis le début.

**L'archive est corrompue** — le téléchargement s'est interrompu ou est incomplet. Re-télécharge le fichier depuis le début.

**Le jeu ne démarre pas** — installe les redistribuables listés ci-dessus, puis essaie en administrateur.

**L'installation plante en cours de route** — dossier de destination plein, RAM insuffisante pendant la décompression, ou fichier exclu de l'antivirus manquant. Vérifie l'espace disque et les exclusions Defender.

**Windows est trop vieux** — certains jeux récents nécessitent Windows 10 22H2 minimum ou Windows 11. Mets Windows à jour.
